# Multi-Class Mental Health Classification

A university project as part of our learning experience in Machine Learning. This is a multi class mental health classification project using TF-IDF and machine learning models

## Overview

This project applies machine learning techniques to classify mental health conditions based on input texts from a Kaggle dataset that was sourced from various social media platforms and other related datasets

## Dataset Details

### Source

- https://www.kaggle.com/datasets/suchintikasarkar/sentiment-analysis-for-mental-health?resource=download

### Variables

- id 
- statement
- status

### Classses

- Anxiety
- Bipolar
- Depression
- Normal
- Personality Disorder
- Stress
- Suicidal

### Sample Size

- 53,000

### Preprocessing Techniques

- Text cleaning
- Convert to lowercase
- Punctuation Removal
- Stopword removal
- Lemmatization
- TF-IDF 

### Models

- Logistic Regression (multinomial)
- Linear SVM
- Multinomial Naive Bayes

### Evaluation Metrics

- Accuracy
- Macro F1-Score
- Weighted F1-Score
